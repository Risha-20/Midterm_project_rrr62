# Midterm Project – CS 634 Data Mining

### Student: Risha R. Dinesh  
### Course: CS 634 – Data Mining (Fall 2025)

This repository contains the complete midterm project implementation for
Frequent Itemset Mining and Association Rule Learning using Brute-Force,
Apriori, and FP-Growth algorithms.

## Repository Structure
src/ – Python & Jupyter source code  
data/ – Transaction datasets (Amazon, Nike, BestBuy, Kmart, General)  
report/ – Final report PDF and screenshots  

## Results Summary
FP-Growth performed fastest across all datasets,
followed by Apriori, while Brute-Force was the slowest but most transparent.
